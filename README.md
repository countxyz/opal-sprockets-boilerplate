# opal-sprockets Boilerplate

## About

Provides a fun little sandbox to have fun with opal. Taken from
(http://opalrb.org/docs/using_sprockets/)

## Start Hacking

```bash
git clone git@github.com:countxyz/opal-sprockets-boilerplate.git
```
